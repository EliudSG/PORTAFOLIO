## Análisis de datos Covid-19 México para pacientes con comorbilidades
Aplicación de metodos de analisis de datos, con uso de herramientas como SQL server, PowerBi y Python con el DataFrame de la última 
actualización de el gobierno de México para el Covid-19
En este proyecto se aplicó:
    - Limpieza de Datos
    - Análisis de datos
    - Modelos predictivos

Esto es parte de una investigación por lo que se requierió hablar con expertos en el area (visto en la presentación adjunta)
