# Proyecto Web
Este es un proyecto web desarrollado utilizando HTML, CSS y JavaScript. El proyecto tiene como objetivo crear una aplicación web interactiva y fácil de usar, se creó con el fin de disminuir la cantidad de hojas gastadas y hacer una hoja de punteo interactiva que de resultados con eficiencia, este es un proyecto pensado en mi comunidad más especificamente un concurso de memorización.
## Características
- Interfaz creada para el usuario, es 100% intuitiva y atractiva.
- Funcionalidades interactivas para mejorar la experiencia del usuario.
- Desarrollada para mejorar el tiempo de calificación y sin necesidad de hacer cálculos humanos evitando posibles errores.
## Tecnologías Utilizadas
- HTML
- CSS
- JS
- Python
